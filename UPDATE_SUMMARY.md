## 🚀 Workflow

1. **Home Page** → Browse products
2. **Add to Cart** → Add multiple items OR
3. **Buy Now** → Direct single item purchase
4. **Cart Page** → Review, modify quantities, remove items
5. **Proceed to Checkout** → Navigate to address.html
6. **Address Form** → Enter delivery details
7. **Select Payment** → Choose payment method
8. **Place Order** → Submit
9. **Success Popup** → Order confirmation with ID
10. **Continue Shopping** → Return to home
